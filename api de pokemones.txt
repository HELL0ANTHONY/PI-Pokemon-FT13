Nota: Para llegar a los detalles del pokemon en particular
Pasos para llegar a los detalles:

1. endpoint 1
2. la url del pokemon te lleva te lleva al segundo endpoint
3. dentro del segundo endpoint existe una propiedad con el nombre: "forms" este es un array dentro de este array existen dos propiedades
uno es el nombre del pokemon y el otro una url. Este url (https://pokeapi.co/api/v2/pokemon-form/1) te lleva a los tipos que pertenece el pokemon
